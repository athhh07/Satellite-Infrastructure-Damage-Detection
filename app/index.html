◈ Architecture

Project Pipeline

Six stages from raw dataset to deployed web application.

01

Dataset preparation

Separate pre/post images from xBD. Convert JSON polygon annotations to grayscale mask PNGs with class IDs 0–4 using Shapely.

02

Preprocessing

Resize to 256×256, ImageNet normalisation, augmentation (flips + rotations), build matched pre/post/mask triplets.

03

Siamese U-Net

Shared encoder encodes both images independently. |post − pre| difference features feed the U-Net decoder as skip connections.

04

Training

Weighted CrossEntropyLoss (background 0.2×, destroyed 4×), Adam, ReduceLROnPlateau, gradient clipping, 30 epochs on Colab T4 GPU.

05

Evaluation

Per-class IoU, mean IoU excluding background, confusion matrix, sample prediction overlays.

06

Streamlit deployment

Upload image pair → inference → colour overlay + per-class stats + CSV/PNG download in a clean dark-themed UI.
